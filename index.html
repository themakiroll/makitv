A library bulletin board on steroids.

MakiTV is a multimedia bulletin board application designed to add spice to current awareness and marketing in any type of library, combining audiovisual media with live Web content to provide a rich information experience. Intuitive, empowering, and open source.

MakiTV is currently in development; the project will be available for forking as soon as it's finished.
